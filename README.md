# personal-site

Small static site built with Next.js and TypeScript to introduce who I am, what I work on, and where to find my projects and writing.

Live: https://akinoztorun.dev

## What it shows

- Short profile and focus as a Senior Frontend Engineer
- Links to GitHub, LinkedIn, and my newsletter "Ownership in Practice"
- Two example projects:
    - Energy portfolio vertical slice (GraphQL, TS, React)
    - Next.js SWR quality demo

## Stack

- Next.js (App Router)
- TypeScript
- Static export, no backend
- Deployed on Vercel

## Local development

```bash
pnpm install   # or npm install / yarn
pnpm dev       # runs on http://localhost:3000
```
